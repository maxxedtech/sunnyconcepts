# Photo Upload Folders

Add your images in this directory tree:

- `portfolio/` → all portfolio images (characterized by default)
- `portfolio/characterized/` → prioritized folder for characterized portfolio images
- `branding/` → logos, brand visuals, identity graphics
- `social/` → social avatar/post-ready images

Tips:
- Use lowercase filenames with dashes (example: `wedding-card-01.jpg`).
- Prefer `.webp` or optimized `.jpg` for large photos.
- Keep transparent assets as `.png` when needed.
